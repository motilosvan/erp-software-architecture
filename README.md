# Taller – Arquitectura ERP (Fábrica de Zapatos)

Este repositorio contiene el desarrollo del taller de documentación de arquitectura usando la plantilla **arc42**, aplicado a un sistema **ERP** para una empresa que **fabrica zapatos** y los provee a empresas comercializadoras (ej: Nike y otras).

El enfoque principal del taller está en el **Módulo de Compras**, el cual permite gestionar proveedores, insumos, órdenes de compra y recepción de materiales para garantizar la continuidad de la producción.

---

## Objetivo del Taller

- Definir épicas e historias de usuario para un ERP.
- Documentar la arquitectura usando **arc42**.
- Modelar diagramas del sistema usando el modelo C4 (C1 y C2).
- Describir un escenario crítico mediante un diagrama de secuencia.
